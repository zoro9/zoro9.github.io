###### 六级标题

段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车。
AAAAAA
BBBBBB
末尾添加两个空格或使用空行换行

字体

Markdown 可以使用以下几种字体：
*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___

显示效果如下

粗体文本
粗体文本
粗斜体文本

Markdown 代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：
`printf()`函数
显示效果：
printf() 函数

代码区块

代码区块使用 4 个空格或者一个制表符（Tab 键）。
‘-‘代表空格
—-<?php
echo ‘hello’;
function test(){
echo ‘test’;
}

显示效果

<?php
echo 'hello';
function test(){
    echo 'test';
}

你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）：
```javascript
$(document).ready(function () {
alert(‘RUNOOB’);
});
```
